# 🎮 Dots & Boxes — v11.7

لعبة Dots & Boxes الكلاسيكية مع ذكاء اصطناعي قوي وموسيقى مصرية أصلية 🏺

---

## 🚀 كيفية اللعب

1. افتح `index.html`
2. اختر حجم اللوحة وعدد اللاعبين
3. اختر وضع اللعب (بشري أو ضد الكمبيوتر)
4. اضغط **"🚀 ابدأ اللعبة"**

---

## ✨ الميزات

- اختيار حجم اللوحة (3×3 حتى 7×7)
- اختيار عدد اللاعبين (2-4)
- معاينة اللوحة قبل البدء
- ألوان مميزة لكل لاعب
- Animations سلسة
- موسيقى خلفية مصرية أصلية مع تحكم كامل بالصوت
- ذكاء اصطناعي بـ 3 مستويات صعوبة

---

## 🤖 مستويات الذكاء الاصطناعي

| المستوى | الوصف |
|---------|-------|
| 😊 سهل | حركات عشوائية مع 30% ذكاء |
| 🧠 متوسط | يكمل المربعات ويتجنب الأخطاء |
| 🔥 صعب | Alpha-Beta Pruning + نظرية السلاسل + حد زمني 800ms |
